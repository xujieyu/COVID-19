# COVID-19

## 项目介绍
这个项目主要是对新型冠状病毒进行一个实实时的监控。
- 数据来自[Wuhan-2019-nCoV项目](https://github.com/canghailan/Wuhan-2019-nCoV)
- 数据通过python脚本进行筛选和切分并放到腾讯云COS上
- 图表采用Echarts绘制
- 支持省级数据和图表
- Vue-cli@4脚手架项目
- [线上示例](http://193.112.19.128)

## 快速开始
安装依赖
```
npm install
```
开发者模式
```
npm run serve
```
项目打包
```
npm run build
```
